
##  功能特性

### 核心功能
-  **题库管理**：支持导入JSON和TXT格式的题库文件
-  **智能答题**：顺序答题和随机答题两种模式
-  **错题记录**：自动记录错题，支持错题回顾和清空
-  **实时统计**：显示答题进度和正确率统计
-  **答案解析**：每道题目提供详细解析
-  **格式转换**：自动将TXT格式题库转换为JSON格式


##  快速开始

### 安装步骤

1. **下载项目文件**
   ```bash
   nisp_quiz_system/
   ├── quiz_system.py      # 主程序
   ├── converter.py        # 题库转换器
   ├── 6练习题六卷.txt     # 示例题库文本文件
   └── 6练习题六卷.json    # 转换后的JSON题库文件
   ```

### 首次使用

1. 运行程序后，点击"加载题库"按钮
2. 选择提供的`6练习题六卷.txt`或`6练习题六卷.json`文件
3. 点击"开始答题"即可开始练习

### 操作流程

#### 1. 加载题库
- 点击"加载题库"按钮
- 选择TXT或JSON格式的题库文件
- 系统会自动处理并显示加载的题目数量

#### 2. 开始答题
- 点击"开始答题"按钮
- 在选项按钮上点击选择答案
- 点击"提交答案"确认选择
- 使用"上一题"/"下一题"按钮导航

#### 3. 查看错题
- 点击"错题本"按钮
- 查看所有错题记录和正确答案
- 可清空错题记录

#### 4. 切换模式
- 点击"随机出题"按钮在顺序/随机模式间切换
- 随机模式下题目顺序会打乱

### 题目格式说明

系统支持两种题库格式：

#### JSON格式
```json
[
  {
    "number": "1",
    "text": "题目内容...",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "answer": "A",
    "explanation": "题目解析..."
  }
]
```

#### TXT格式
```
1．题目内容...
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：A
解析：题目解析...
```

### 创建题库

   ```python

   ```

2. **手动创建JSON题库**
   - 参考示例格式创建JSON文件
   - 确保包含题目、选项、答案、解析四个部分
   - 支持单选题和多选题（答案如"A"、"AB"等）

### 题库要求
- **单选题**：答案为一个字母（如"A"、"B"）
- **多选题**：答案为多个字母组合（如"AB"、"ACD"）
- 每个题目必须包含完整的四个选项
- 解析部分可以为空

### 日志查看
- 错题记录保存在`wrong_questions.json`文件中
- 转换后的题库文件与原TXT文件在同一目录


### 自定义设置
可通过修改`QuizSystem`类中的颜色配置自定义界面：
```python
self.colors = {
    'bg': '#f0f0f0',           # 背景色
    'question_bg': '#ffffff',   # 题目区域背景色
    'button_bg': '#4CAF50',     # 按钮背景色
    'button_fg': '#ffffff',     # 按钮文字颜色
    'wrong_bg': '#ffebee',      # 错误答案背景色
    'correct_bg': '#e8f5e9'     # 正确答案背景色
}
```
